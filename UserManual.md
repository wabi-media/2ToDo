## Getting Started

### First Launch
When you first open 2ToDo, you'll see:
- A beautiful themed background (Ocean Blue by default)
- The main list view showing "My Lists" and "Shared with Me" sections
- A floating "+" button to create your first list

### Permissions
For the best experience, grant these permissions when prompted:
- **Location Services**: Required for location-based reminders
- **Notifications**: Get alerts when you arrive at locations or deadlines approach
- **iCloud**: Enable seamless sync across your devices and sharing with others

## Creating Lists

### Create a New List

1. **Tap the "+" button** at the bottom right of the screen
2. **Enter a list name** (e.g., "Grocery Shopping", "Home Repairs")
3. **Add a location** (optional but recommended):
   - Tap "Pick on Map" to select any location
   - Tap "Use Current Location" to use where you are now
   - Choose from your "Favorite Locations" if you've saved any
   - Set the trigger radius (25-500 meters) - this determines how close you need to be to get notified
4. **Add notes** (optional): Additional context or details about the list
5. **Set a deadline** (optional): Get reminded before time runs out
6. **Tap "Create"** to save your list

### List Types
- **Local Lists**: Stored only on your device
- **Shared Lists**: Synced via iCloud and accessible to multiple users
- **Archived Lists**: Completed lists you want to keep for reference

## Managing Todo Tasks

### Adding Tasks to a List

1. **Open a list** by tapping on it
2. **Tap the "+" button** at the bottom
3. **Enter the task name** (e.g., "Buy milk", "Fix leaky faucet")
4. **Tap "Add"** to save

### Completing Tasks

- **Tap the circle** next to a task to mark it complete
- Completed tasks show a checkmark and move to the bottom
- **Tap again** to mark incomplete

### Editing Tasks

1. **Tap on a task** to edit it
2. **Modify the text**
3. **Tap "Save"** to update

### Deleting Tasks

1. **Swipe left** on a task
2. **Tap the trash icon** to delete

### Reordering Tasks

1. **Long press** on a task
2. **Drag** it to the desired position
3. **Release** to drop

### Ignoring Tasks

Some tasks you might want to skip without deleting:
1. **Swipe left** on a task
2. **Tap the eye icon** to ignore
3. Ignored tasks are dimmed and don't count toward list completion

## Location-Based Features

### How Location Reminders Work

When you add a location to a list:
- 2ToDo creates a **geofence** (virtual boundary) around that location
- When you **enter the radius**, you get a notification
- The notification shows which tasks are on that list
- Perfect for remembering things when you're at the store, office, or home!

### Setting Up Location Reminders

1. **Create or edit a list**
2. **Add a location** using one of these methods:
   - **Pick on Map**: Browse and select any location
   - **Current Location**: Use where you are right now
   - **Favorites**: Choose from previously saved locations
3. **Adjust the radius** (25-500m):
   - Smaller radius: More precise, triggers when very close
   - Larger radius: Triggers earlier, good for busy areas
4. **Save the list**

### Editing a List's Location

1. **Open the list**
2. **Tap the pencil icon** next to the location card
3. **Modify** the location, name, or radius
4. **Tap "Save"** to update

### Managing Favorite Locations

**Adding to Favorites:**
1. When picking a location, tap the **star icon**
2. Name your favorite location
3. It's now available for quick selection on future lists

**Viewing Favorites:**
- Open Settings ‚Üí Locations
- See all your favorite locations
- Tap to view on map or delete

### Location Permissions

**Always Allow** (Recommended):
- Get notifications even when the app is closed
- Most reliable for location reminders

**While Using**:
- Only works when app is open
- Limited notification capability

## Sharing & Collaboration

### Sharing a List with Others

1. **Open the list** you want to share
2. **Tap the share icon** (person with plus) in the top right
3. **Choose sharing permission**:
   - **Anyone with Link**: Anyone who has the link can access
   - **Specific Users**: Only invited people can access
4. **Tap "Start Sharing"** or "Invite People"
5. **Share the link** via:
   - Messages
   - Mail
   - AirDrop
   - Copy link to share anywhere

### Accepting a Shared List

1. **Tap the share link** you received
2. **Tap "Accept"** in the prompt
3. The list appears in your "Shared with Me" section
4. You can now view and edit tasks (if permitted)

### Managing Shared Lists

**As the Owner:**
- **View participants**: See who has access in the Share screen
- **Remove participants**: Tap the X next to a participant's name
- **Change permissions**: Switch between "Anyone" and "Specific Users"
- **Stop sharing**: Tap "Stop Sharing" to revoke all access

**As a Participant:**
- **View and edit**: Add, complete, or delete tasks
- **See changes in real-time**: Updates from others appear automatically
- **Leave share**: Tap "Leave Share" to remove yourself from the list
- **Get notifications**: See when others add or complete tasks

### Collaboration Features

**Real-Time Sync:**
- Changes appear instantly for all participants
- See who made the last change
- Conflict resolution happens automatically

**Notifications:**
- Get notified when someone adds a task
- See when tasks are completed by others
- Know when someone shares a list with you

**Activity Feed:**
- View recent collaboration events
- See who added, completed, or deleted tasks
- Track changes to shared lists

## Deadlines & Notifications

### Setting a Deadline

1. **Open a list**
2. **Tap the calendar icon** or "Add Deadline" section
3. **Toggle "Has Deadline" on**
4. **Select date and time**
5. **Tap "Save"**

### Deadline Notifications

You'll receive notifications:
- **1 week before** the deadline
- **1 day before** the deadline
- **1 hour before** the deadline
- **At the deadline** time

### Editing or Removing Deadlines

1. **Open the list**
2. **Tap the deadline section**
3. **Modify the date/time** or toggle off to remove
4. **Tap "Save"**

### Notification Types

**Location Notifications:**
- Triggered when you enter a geofence
- Shows list name and task count
- Tap to open the list

**Deadline Notifications:**
- Remind you of upcoming deadlines
- Show time remaining
- Tap to view the list

**Collaboration Notifications:**
- Someone shared a list with you
- Tasks added/completed by others
- Changes to shared lists

**Achievement Notifications:**
- Daily goal completed
- Weekly goal achieved
- Milestones reached

## Siri & Voice Commands

Use your voice to manage tasks hands-free! 2ToDo integrates with Siri for quick task management.

### Getting Started with Siri

**Enable Siri Shortcuts:**
1. Open the **Shortcuts app** on your iPhone
2. Search for **2ToDo**
3. You'll see available shortcuts ready to use
4. Tap a shortcut to add it or customize the trigger phrase

### Voice Commands

**Add a Task:**
Say: *"Hey Siri, add a task to 2ToDo"*

Siri will ask you:
- What task do you want to add?
- Which list should it go to?

**Examples:**
- "Hey Siri, add buy milk to 2ToDo"
- "Hey Siri, add a task to my grocery list in 2ToDo"
- "Hey Siri, add pick up dry cleaning tomorrow to 2ToDo"

---

**Create a New List:**
Say: *"Hey Siri, create a 2ToDo list"*

Siri will ask you:
- What should the list be called?

**Examples:**
- "Hey Siri, create a new 2ToDo list called Weekend Errands"
- "Hey Siri, create a shopping list in 2ToDo"

---

**Show Your Lists:**
Say: *"Hey Siri, show my 2ToDo lists"*

This opens the app and shows a summary of your lists.

**Examples:**
- "Hey Siri, show my 2ToDo lists"
- "Hey Siri, open 2ToDo"

---

**Complete a Task:**
Say: *"Hey Siri, complete a task in 2ToDo"*

Siri will ask you:
- Which task do you want to complete?
- (Optionally) Which list is it in?

**Examples:**
- "Hey Siri, mark buy milk as done in 2ToDo"
- "Hey Siri, complete the laundry task in 2ToDo"

### Natural Language Support

When adding tasks, 2ToDo understands natural language:
- **Dates**: "buy milk tomorrow", "call mom on Friday"
- **Times**: "meeting at 3pm", "dentist at 10:30"
- **Deadlines**: "finish report by next Monday"

## Apple Watch Integration

### Viewing Lists on Watch
1. Open **2ToDo** on your Apple Watch
2. Browse your lists with a beautiful frosted glass interface
3. Tap a list to see tasks, deadlines, and location info
4. Lists are sorted by **Location** (nearest first), **Deadline**, and then **Importance**

### Completing Tasks on Watch
1. Open a list
2. Tap a task to toggle completion
3. Changes sync automatically to your iPhone
4. Use the "Clear Done" button to tidy up your list

### Watch Complications (Widgets)

Customize your watch face with 2ToDo complications:

- **Circular**: Shows list progress ring and remaining count
- **Corner**: Elegant curved text showing total pending tasks (e.g., "5")

### Sync Status
- **Independent Operation**: Watch app works even if iPhone is away (uses cached data)
- **Background Sync**: Updates smoothly in the background
- **Theme Sync**: Your beautiful app theme carries over to the Watch!

## Data Management

### Backup & Restore
Keep your data safe with our robust backup system.

**Creating a Backup:**
1. Go to **Settings** ‚Üí **Backup & Restore**
2. Tap **Create Backup**
3. Choose **Local Backup** (on device) or **iCloud Backup** (safe in cloud)
4. Backups include all lists, items, themes, and settings

**Restoring Data:**
1. Go to **Settings** ‚Üí **Backup & Restore**
2. Tap on any existing backup file
3. Select **Restore**
4. Confirm to overwrite current data with the backup

### Exporting Data
**Export All Lists:**
1. Go to **Settings**
2. Tap **Export All Lists**
3. Choose where to save the JSON file
4. Share via AirDrop, Mail, or save to Files

**Export Single List:**
1. Open the list
2. Tap the **share icon** (square with arrow)
3. Choose **Export**
4. Save or share the file

### Importing Data
1. Go to **Settings**
2. Tap **Import Lists**
3. Select a previously exported JSON file
4. Lists are imported with new IDs to avoid conflicts

### Archiving Lists

**Archive a Completed List:**
1. Open the list
2. Tap the **archive icon** (box)
3. The list moves to "Archived" section

**View Archived Lists:**
1. Go to **Settings** ‚Üí **Archived Lists**
2. Tap a list to view
3. Tap **Unarchive** to restore

**Delete Archived Lists:**
1. Go to **Settings** ‚Üí **Archived Lists**
2. Swipe left on a list
3. Tap **Delete**

### Deleting Lists

**Delete a Single List:**
1. On the main screen, swipe left on a list
2. Tap the **trash icon**
3. Confirm deletion

**Note:** 
- Deleting a shared list you own will remove it for all participants!
- If you delete a list that is part of a **Group**, you will receive a specific warning to confirm its removal from those groups.

## Technical Support

### Common Questions

**Q: Why isn't my widget updating?**
A: Widgets refresh periodically. Open the main app to force an immediate refresh if you just made changes.

**Q: "No List Selected" on Widget?**
A: Long press the widget, choose "Edit Widget", and select the specific list you want to track.

**Q: Location alerts not working?**
A: Ensure you have granted "Always Allow" location permissions so 2ToDo can detect geofences even when closed.

**Q: Backup failed?**
A: Check your available iCloud storage. For local backups, ensure you have device space.

## Privacy & Security

### Data Storage

- **Local data**: Stored securely on your device
- **iCloud data**: Encrypted in transit and at rest
- **Shared lists**: Only accessible to invited participants

### Permissions

- **Location**: Only used for geofencing, never tracked
- **Notifications**: Only sent for your lists and deadlines
- **Contacts**: Never accessed or stored

### Sharing

- **Share links**: Can be revoked at any time
- **Specific users**: More secure than "Anyone with link"
- **Participant management**: You control who has access
